# Explainable Alzheimer’s Detection from MRI using ResNet & Grad-CAM

## 🧠 Overview
This project presents an explainable deep learning approach for Alzheimer’s disease detection using brain MRI scans. A ResNet-based CNN is combined with Grad-CAM to provide transparent and clinically interpretable predictions.

## 🎯 Objectives
- Classify MRI scans into Alzheimer’s-related categories
- Provide visual explanations using Grad-CAM
- Improve trust and interpretability in medical AI systems

## 🛠 Built With
- Python
- PyTorch & Torchvision
- ResNet-18
- Grad-CAM
- OpenCV
- NumPy, Pandas
- Scikit-learn
- Matplotlib, Seaborn
- Kaggle (GPU)

## 📊 Results
- Strong validation and test accuracy
- Balanced class-wise performance
- Grad-CAM highlights disease-relevant brain regions

## 🔍 Explainability
Grad-CAM is applied to the final convolutional layer of ResNet to visualize regions that most influence model predictions, ensuring transparency in medical decision-making.
